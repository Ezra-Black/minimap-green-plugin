# Green Screen

Adds a greenscreen, useful for content creators. Original author: I Yam Jeremy

**Current maintainer:** Ezra-Black  
**Previous maintainer(s):** Trevor159

This plugin is maintained in this repository ([minimap-green-plugin](https://github.com/Ezra-Black/minimap-green-plugin)).

## Modes

- **Full game only** – Greenscreen over the game world (player model cut out)
- **Minimap only** – Greenscreen over the minimap circle (for streaming)
- **Both** – Full game and minimap

## Config

- **Color** – Greenscreen color (for OBS/Streamlabs chroma key)
- **Toggle key** – Keybind to show/hide
- **Should default on** – Start with greenscreen visible or hidden

## Build & run

```bash
./gradlew run
```

Windows: `.\gradlew.bat run` or `.\run-developer.ps1`
